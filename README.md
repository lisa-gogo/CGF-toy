# CGF-toy </br>
This is  A Small Calculator using Context-Free Grammar  </br>
It includes +-*/ ( multiplication, Minus, addition , Division) </br>
As for CGS, basic idea is blow : </br>

Start variable is e. </br>
e -> e + t | e-t | t     </br> 
t -> t x f | t / f | f</br>
f -> (e) | number   #bracket </br>
